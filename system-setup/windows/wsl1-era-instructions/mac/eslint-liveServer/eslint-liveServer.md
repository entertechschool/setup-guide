# Instala ESLint y Live-Server

**Ejecuta:**

`npm -g i eslint git-open live-server`

Obtendrás bastante feedback mientras se instala.

Después de que se haya completado, **ejecuta:**

`eslint --version`

Deberás ver algo parecido a:

```
v6.7.2
```

**ejecuta:**

`live-server --version`

Deberás ver algo parecido a:

```
live-server 1.2.1
```
