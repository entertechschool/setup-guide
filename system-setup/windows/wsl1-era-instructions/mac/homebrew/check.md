#  Revisa el Homebrew

Ejecuta:

`brew doctor`

**Si** el output es: 
```
Your system is ready to brew
``` 
**ejecuta:**

`brew update` 

 [Continúa con el siguiente paso](../tree/tree.md)

 ---

**O si** el output es: 
```
-bash: brew: command not found
```
 
 **entonces** [haz click aquí para instalar Homebrew](../homebrew/homebrew.md)
