# Instalar Tree

**Ejecuta:**

 `brew install tree`

Verifica que tree está instalado **ejecutando:**

 `tree --version`

Output esperado: 

```
tree v1.8.0 (c) 1996 - 2018 by Steve Baker, Thomas Moore, Francesc Rocher, Florian Sesser, Kyosuke Tokoro
```
